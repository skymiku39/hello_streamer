# Hello Streamer v0.9.1

## 介面與行為修正

### 多國語言文字不再被裁切

切換英文、日文、韓文等較長文案時，主視窗工具列按鈕、底部狀態列、觸發行為選單、頻道列狀態標籤，以及各對話框（新增頻道、語言、瀏覽器設定）的按鈕，會依目前語言的字寬自動調整寬度，避免 CustomTkinter 固定寬度造成的省略號裁切。主視窗最小寬度略調整為 920px，以容納較長的工具列文字。

### 只監測模式不再誤關播放器

「只監測」模式下，離線或離題時不再執行自動關閉播放器視窗，與文件說明一致——此模式僅更新狀態，不觸發開啟或關閉瀏覽器。

### 瀏覽器視窗追蹤更精準

啟動後追蹤 HWND 時會再次比對標題關鍵字，避免把尚未載入直播標題的空白視窗誤登記為已追蹤。

## 下載檔案

- Windows 請下載 `HelloStreamer-v0.9.1-windows-x64.exe`
- Linux x64 請下載 `HelloStreamer-v0.9.1-linux-x64.tar.gz`
- Raspberry Pi 64-bit 請下載 `HelloStreamer-v0.9.1-linux-arm64.tar.gz`

## 升級提醒

- 從 v0.9.0 升級可直接覆蓋執行檔；`config.json` 與 Profile 資料夾無需變更。
- 若你依賴「只監測」同時保持播放器開啟，v0.9.1 起離線時將不再自動關窗，這是預期行為。
